RKTAPF is a Admin Page Finder tool.Its build on python program.


### Setup ###


+ sudo pip install -r requirements.txt


### Terminal Command ###

+ git clone https://github.com/r3k4t/rktapf.git
+ cd rktapf
+ sudo python3 rktapf.py
